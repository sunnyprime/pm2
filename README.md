"# pm2" 

## pm2 command list:


#### List All Running Processes
``` pm2 ls ```

#### View Logs of All Processes
``` pm2 log ```

#### View Logs of a Specific Process
``` pm2 log id/name ```

#### Start a Process
``` pm2 start app.js ```

#### List Running Processes
``` pm2 list ```

#### Stop a Process
``` pm2 stop id/name ```

#### Start a Previously Stopped Process
``` pm2 start id/name ```

#### Restart a Process
``` pm2 restart id ```

#### Kill a Specific Process
``` pm2 kill id/name ```

#### Kill All Processes
``` pm2 kill ```

#### Generate Ecosystem Configuration
``` pm2 ecosystem ```
